# N Queens Solver

The N Queens Solver is a Python program that solves the N queens puzzle. The challenge is to place N non-attacking queens on an NÃ—N chessboard.

## Requirements

- Python 3.4.3 or higher
- PEP 8 style (version 1.7.*)
- sys module (for argument handling)
